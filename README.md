# Sales Performance Analysis System

## Project Overview
This project analyzes retail sales data using Python. It performs data cleaning, calculates sales metrics, generates charts, and creates a sales report to provide meaningful business insights.

## Features
- Load sales dataset from a CSV file
- Clean missing and duplicate data
- Calculate total sales and quantity sold
- Identify top-selling products
- Perform category-wise and monthly sales analysis
- Generate Bar, Line, and Pie charts
- Create a sales report (`report.txt`)

## Technologies Used
- Python
- Pandas
- NumPy
- Matplotlib
- Microsoft Excel
- SQL (Basic Concepts)

## How to Run
1. Install the required libraries:
   ```bash
   pip install pandas numpy matplotlib
   ```
2. Place `sales_data.csv` in the project folder.
3. Run the program:
   ```bash
   python main.py
   ```

## Output
The project generates:
- Sales summary
- Total sales and quantity sold
- Top-selling products
- Category-wise sales analysis
- Monthly sales analysis
- Bar chart
- Line chart
- Pie chart
- Sales report (`report.txt`)
